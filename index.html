GaggiMate Pump Flow Calibration

Full automation of the 1 bar / 9 bar pump flow calibration. Loads the profile, runs the shot, analyzes the history, and updates the coefficients — without leaving the machine's UI.

Install (one time)

  1. Drag the button below to your browser's bookmarks bar.
  2. Open your GaggiMate web UI (e.g. http://gaggimate.local).
  3. Click the bookmarklet. A panel appears with a Start calibration button.
🔧 Calibrate GaggiMate

The bookmarklet carries all the logic. Once installed it doesn't depend on this page, and it doesn't send anything to the internet.

What it does

  1. Reads the machine's current coefficients (pumpModelCoeffs).
  2. Imports and selects the [Calibration] Pump flow calibration profile.
  3. Switches to BREW mode and starts the shot (you need to adjust the steam valve so pressure reaches 1 bar and then 9 bar).
  4. Waits for the shot to finish, downloads the latest .slog, extracts samples from the Measure phases.
  5. Computes new coefficients (actual flow / estimated flow × current coeff.) and lets you save them back via POST /api/settings.
Fallback: analyze a downloaded file manually
